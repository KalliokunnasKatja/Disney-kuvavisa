# Disney-kuvavisa

Disney-kuvavisa on Sveltellä toteutettu selainpohjainen kuvavisa, joka on suunnattu Disney-aiheesta kiinnostuneille käyttäjille. Sovellus toimii kevyenä ajanvietepelinä ja perustuu hahmojen tunnistamiseen kuvien perusteella.

## Tarkoitus

Projektin tavoitteena oli toteuttaa yksinkertainen ja viihteellinen arvauspeli sekä harjoitella Svelte-kehitystä, ulkoisten rajapintojen hyödyntämistä ja käyttöliittymäkomponenttien käyttöä.

## Teknologiat

- Svelte
- JavaScript
- HTML & CSS

## Typografia

Sovelluksen typografiassa on käytetty Google Fonts -kirjaston **Sedgwick Ave Display** -fonttia, joka tukee pelin leikkisää ja sarjakuvamaista ilmettä.  
https://fonts.google.com/specimen/Sedgwick+Ave+Display

## Rajapinnat

Pelissä käytettävät hahmot ja niihin liittyvät tiedot haetaan **Disney API** -rajapinnan kautta.  
https://disneyapi.dev/

## Ulkoiset kirjastot ja komponentit

- **AgnosticUI**  
  - Modal-ikkunan sulkemispainike  
  - Aloitusnäkymässä näkyvä latausanimaatio (loader)  
  https://www.agnosticui.com/
